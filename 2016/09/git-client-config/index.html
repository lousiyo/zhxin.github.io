Git客户端初始配置

2016/09/19

  • 安装完成GIT之后配置相关用户名邮箱.

    git config --global user.name "zhxin"
git config --global user.email zhxin@outlook.com

  • 配置连接github.

    ls -al ~/.ssh #查看是否生成SSH key
ssh-keygen -t rsa -b 4096 -C "zhxin@outlook.com" #生成key
eval "$(ssh-agent -s)" #保证ssh可用
ssh-add ~/.ssh/id_rsa #添加ssh key到客户端

  • 将ssh key添加到github账户

    clip < ~/.ssh/id_rsa.pub
ssh -T git@github.com #测试是否连接到github
