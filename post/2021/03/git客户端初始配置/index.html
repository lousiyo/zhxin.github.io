Git客户端初始配置

2021-03-28 git 139 字 1 分钟
  • 安装完成GIT之后配置相关用户名邮箱.
1
2

git config --global user.name "xxx"
git config --global user.email xxx@xxx.com
  • 配置连接github.
1
2
3
4

ls -al ~/.ssh #查看是否生成SSH key
ssh-keygen -t rsa -b 4096 -C "xxx@xxx.com" #生成key
eval "$(ssh-agent -s)" #保证ssh可用
ssh-add ~/.ssh/id_rsa #添加ssh key到客户端
  • 将ssh key添加到github账户
1
2

clip < ~/.ssh/id_rsa.pub
ssh -T git@github.com #测试是否连接到github
